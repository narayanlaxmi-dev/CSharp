LINQ in C# - Simple Explanations and Examples

What is LINQ?

LINQ (Language Integrated Query) is a feature in C# that allows you to query collections like arrays, lists, or databases in a readable and concise way.

1. Filtering with where


var numbers = new List<int> { 1, 2, 3, 4, 5 };
var evenNumbers = numbers.Where(n => n % 2 == 0);

foreach (var num in evenNumbers)
{
    Console.WriteLine(num);
}

2. Sorting with orderby


var names = new List<string> { "Alice", "Bob", "Charlie" };
var sortedNames = names.OrderBy(name => name);

foreach (var name in sortedNames)
{
    Console.WriteLine(name);
}

3. Selecting with select


var numbers = new List<int> { 1, 2, 3 };
var squares = numbers.Select(n => n * n);

foreach (var square in squares)
{
    Console.WriteLine(square);
}

4. Combining with select and anonymous types


var people = new List<Person>
{
    new Person { Name = "John", Age = 30 },
    new Person { Name = "Jane", Age = 25 }
};

var namesWithAges = people.Select(p => new { p.Name, p.Age });

foreach (var item in namesWithAges)
{
    Console.WriteLine($"{item.Name} is {item.Age} years old.");
}

Note: Make sure you have using System.Linq; at the top of your file to use LINQ.
